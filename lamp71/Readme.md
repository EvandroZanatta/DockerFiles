# LAMP 7.1

Ambiente LAMP para desenvolvimento com PHP 7.1

## Itens instalados

- Apache 2.4 ([#ssl](instalar SSL))
- PHP 7.1
- MySQL 5.7 (root:123456)
- phpMyAdmin

## Recursos extras

- Node.js 8
- MC (Editor)
- Composer
- Git

## Portas abertas

- 80
- 443

## Ambiente de trabalho

- /var/www/html
